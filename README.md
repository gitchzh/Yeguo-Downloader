# 椰果视频下载器 v1.0.1

一个简单、免费、无套路的视频下载工具，支持从 YouTube 和 Bilibili（B站）下载视频。

## ✨ 功能特性

### 🎯 核心功能
- **多平台支持**：支持 YouTube 和 Bilibili 视频下载
- **批量下载**：支持同时下载多个视频
- **格式选择**：提供多种分辨率和格式选择
- **智能解析**：自动识别视频格式和最佳下载选项

### 🎨 用户界面
- **现代化UI**：采用 Cursor 风格的浅色主题
- **中文界面**：完全中文化的用户界面
- **响应式设计**：支持不同屏幕尺寸
- **实时状态**：下载进度和状态实时显示

### ⚡ 下载功能
- **并发下载**：支持多线程并发下载
- **断点续传**：支持下载中断后继续下载
- **速度控制**：可设置下载速度限制
- **格式转换**：支持视频格式自动转换

### 🔧 高级功能
- **设置管理**：完整的应用程序设置系统
- **日志管理**：详细的日志记录和查看功能
- **问题反馈**：用户反馈问题，自动发送邮件（支持Gmail和QQ邮箱）
- **路径管理**：灵活的下载路径设置
- **文件命名**：自定义文件命名规则
- **快捷键支持**：完整的键盘快捷键操作

## 🚀 快速开始

### 系统要求
- **操作系统**：Windows 10/11
- **Python版本**：Python 3.8 或更高版本
- **内存要求**：至少 4GB RAM
- **网络要求**：稳定的网络连接

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/gitchzh/Yeguo-Downloader.git
   cd yeguo-downloader
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 使用说明

1. **启动程序**：运行 `python main.py` 启动应用程序
2. **输入链接**：在输入框中粘贴视频链接（支持批量输入）
3. **解析视频**：点击"解析"按钮获取可用格式
4. **选择格式**：在格式列表中选择需要的分辨率
5. **开始下载**：点击"下载选中项"开始下载
6. **问题反馈**：使用"帮助" → "问题反馈"提交问题

### 界面功能
- **菜单栏**：提供文件、编辑、工具、帮助等功能
- **工具栏**：快速访问常用功能
- **状态栏**：显示当前操作状态和进度
- **设置面板**：配置下载路径、并发数等参数

### 反馈功能配置

如需使用问题反馈功能，请按照以下步骤配置邮件：

1. **快速配置**：参考 [QUICK_SETUP.md](QUICK_SETUP.md) 快速配置指南
2. **运行配置脚本**：`python setup_email.py`
3. **按照提示配置**：输入邮箱地址和应用密码/授权码
4. **激活配置**：运行生成的配置脚本
5. **测试功能**：运行 `python test_email.py` 测试配置
6. **使用功能**：在程序中点击"帮助" → "问题反馈"

详细配置说明请参考 [EMAIL_SETUP.md](EMAIL_SETUP.md) 文档。

## 🎯 支持格式

### 视频分辨率
- **4K**：3840x2160, 4096x2160
- **2K**：2560x1440, 2048x1080
- **1080P**：1920x1080, 1440x1080
- **720P**：1280x720, 960x720
- **480P**：852x480, 640x480
- **360P**：640x360, 480x360
- **240P**：426x240
- **144P**：256x144

### 输出格式
- **MP4**：最常用的视频格式
- **MKV**：高质量容器格式
- **WebM**：Web优化格式
